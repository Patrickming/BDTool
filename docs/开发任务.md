## ⚠️ 注意事项

1. **安全性：**
   - 不要自动发送 DM（避免 Twitter 封号）
   - 密码必须加密存储（bcrypt）
   - JWT Secret 至少 32 个字符
   - 所有输入必须验证（Zod）

2. **代码规范：**
   - 所有代码注释使用中文
   - 遵循 ESLint 和 Prettier 规则
   - 使用 TypeScript 路径别名（@/）

3. **Git 提交：**
   - 每个功能一个 commit
   - commit message 使用中文
   - 包含详细说明
